Hi, I'm Johnny Nguyen, a first-generation CS student from Fort Wayne, Indiana, studying Computer Science on the Systems Programming track at Purdue. I write a lot of C and C++, and I'm most interested in systems programming. I have built low-level systems things like a Unix shell, a malloc implementation, and an HTTP server from scratch.

This summer I'm interning at Sandia National Laboratories in Albuquerque, NM, working on dataflow architecture simulation by modeling how general-purpose programs execute on spatial accelerators and exploring instruction-level optimizations using the Structural Simulation Toolkit (SST).

I also do undergraduate research with RTX through Purdue's The Data Mine, simulating RF signals for downstream classification models, and occasionally I build things for fun. For example, I have made a Bad Apple!! renderer in spectrograms.

This site is where I post things I find interesting. Check GitHub or LinkedIn for more of my work.
